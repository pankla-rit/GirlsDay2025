Interaktive Aufgaben für HTML, CSS & JS

Komm gerne auf mich zu, falls du Fragen hast!

Aufgabe 1: Erstelle einen Absatz

Beschreibung: Erstelle einen Absatz mit dem <p>-Tag.

Beispielcode:

<p>Hello GirlsDay!</p>

Erklärung: Das <p>-Tag definiert einen Absatz.

Rätsel: Füge in deinen Absatz das versteckte Wort "Blume" ein.

Aufgabe 2: Erstelle eine Überschrift

Beschreibung: Erstelle eine Überschrift der Ebene 1 mit dem <h1>-Tag.

Beispielcode:

<h1>Willkommen!</h1>

Erklärung: Überschriften helfen, Inhalte zu strukturieren.

Rätsel: Schreibe in deine Überschrift die magische Zahl "1" mit hinein.

Aufgabe 3: Erstelle einen Link

Beschreibung: Erstelle einen Hyperlink mit dem <a>-Tag.

Beispielcode:

<a href="https://example.com">Besuche Example</a>

Erklärung: Mit dem <a>-Tag erstellt man Links.

Rätsel: In "exmple.com" fehlt ein Buchstabe – füge ihn ein, damit es "example.com" wird.

Aufgabe 4: Füge ein Bild ein

Beschreibung: Erstelle ein Bild-Element mit dem <img>-Tag. Nimm dafür gerne die URL: https://shorturl.at/W4S6g.

Beispielcode:

<img src="Bild URL">

Erklärung: Das <img>-Tag bindet ein Bild ein.

Rätsel: Ergänze die URL im Code.

Aufgabe 5: Ändere die Hintergrundfarbe

Beschreibung: Ändere mit Inline-CSS die Hintergrundfarbe eines Elements.

Beispielcode:

<div style="background-color: pink;">Inhalt</div>

Erklärung: Mit style="background-color: ..." kannst du das Styling direkt im HTML-Tag festlegen.

Rätsel: Finde den Hex-Code für Rosa. (Tipp: #FFC0CB)

Vorschaubereich

Aufgabe 6: Ändere die Schriftgröße

Beschreibung: Ändere mit Inline-CSS die Schriftgröße eines Textes. Wir brauchen nur den CSS Befehl und die Größe.

Beispielcode:

<p style="font-size: 24px;">Großer Text</p>

Erklärung: Mit font-size legst du die Textgröße fest.

Rätsel: Erhöhe 24px um 50%. Welche Größe ergibt das?

Vorschaubereich

Aufgabe 7: Ändere den Text mit JavaScript

Beschreibung: Nutze JavaScript, um den Text eines Elements zu ändern.

Beispielcode:

document.getElementById('task7-preview').innerText = 'Hallo GirlsDay!';

Erklärung: Mit document.getElementById() erreichst du ein Element und kannst seinen Inhalt ändern.

Rätsel: Ändere "Hallo" in "Hi" und füge ein Ausrufezeichen hinzu.

Vorschaubereich

Aufgabe 8: Erstelle eine Variable

Beschreibung: Erstelle eine Variable und gib ihren Wert aus.

Beispielcode:

let greeting = 'Hallo GirlsDay!';
console.log(greeting);

Erklärung: Variablen speichern Daten. Mit console.log() kannst du sie ausgeben.

Rätsel: Ändere den Text in Großbuchstaben. (Tipp: .toUpperCase())

Schau in die Browser-Konsole (F12)

Aufgabe 9: Erstelle eine Funktion

Beschreibung: Schreibe eine Funktion, die eine Nachricht in der Konsole ausgibt.

Beispielcode:

function sayHello() {
  console.log('Hallo!');
}
sayHello();

Erklärung: Funktionen führen Codeblöcke aus, wenn sie aufgerufen werden.

Rätsel: Ändere die Funktion so, dass sie deinen Namen ausgibt.

Schau in die Browser-Konsole (F12)
